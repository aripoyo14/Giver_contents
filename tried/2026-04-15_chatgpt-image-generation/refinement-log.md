## 最終稿

```
【今日のテック/AIニュース】

📌 欲しい画像、生成AIで作れます

ChatGPT・Gemini・Copilotなど、多くの生成AIでイラストや画像を作れます。「壁を乗り越える場面を横から見たアイコン、背景は白」のように場面を文章で説明するだけ。使うモデルによって仕上がりに差がありますが、プロンプトの工夫で精度を上げられます。

💬 実際に試してみた——講義資料に使うイラストを作るために、「壁を3人で乗り越える場面のアイコン、背景は白」と入力してみました。ChatGPTで出てきたのは白黒の棒人間……全然違う！同じプロンプトをGeminiに入れると、最初からそこそこいい感じのものが出てきました。ChatGPTには「色を入れて、アニメーション風のテイストになるようにプロンプトを改良して」と指示し、その改良版プロンプトで試したら使えるレベルに。どのAIを使うにしても、プロンプト次第で結果がかなり変わります。

資料のイラストをフリー素材サイトで探すのに時間がかかっている人——生成AIに説明を入力するだけで、自分専用の画像が作れます。思いがけない仕上がりになるかも。まず1枚、試してみてください！
```

## ブラッシュアップから得た知見

1. **一発でできる誇張はしない** → 「数ターン必要」という正直さが信頼感につながる
2. **ツール名は複数挙げる** → 「社内で使えるもので試せる」と感じさせる
3. **「何が違ったか」の具体描写が刺さる** → 白黒の棒人間という具体例が読者の共感を生む
4. **プロンプト改良の方向性を示す** → 「テイスト・色味」という切り口が読者の参考になる
5. **文体はありちゃんの言葉に近づける** → 生成AIっぽい丁寧さより、短文＋感情表現が読みやすい
6. **締めは「試してみてください」で背中を押す** → 情報提供で終わらず行動を促す
