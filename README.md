# Tech0 Giver Contents Pipeline

Tech0 LINE 登録者向け「ギブ型」コンテンツの収集・ドラフト生成パイプライン。

テック/AI の最新ニュースを初学者向けに噛み砕き、LINE メッセージとして配信するためのコンテンツを半自動で作成する。

## 全体フロー

```
RSS フィード取得（Python / Cron 毎朝8時）
  ↓
inbox/ に記事一覧を Markdown で保存
  ↓
Claude Code コマンドで 4軸スコアリング → LINE ドラフト10案を生成
  ↓
drafts/ に保存 → GitHub PR でレビュー
  ↓
承認後、LINE へ手動配信
```

## 記事収集方針

### ソース Tier 定義


| Tier             | 役割                              | ソース                                                                                   | 重み      |
| ---------------- | ------------------------------- | ------------------------------------------------------------------------------------- | ------- |
| **Tier 1（一次情報）** | コンテンツの主情報源                      | TechCrunch AI, MIT Technology Review, The Verge AI, OpenAI Blog, Google DeepMind Blog | 0.8〜1.0 |
| **Tier 2（表現参考）** | 文体・平易さの参考としてのみ使用。ドラフトには採用しない    | Qiita AI タグ, Zenn トレンド                                                                | 0.4     |
| *Tier 1.5（将来）*   | 海外インフルエンサー X 投稿（X API Basic 必要） | V2 で追加予定                                                                              | —       |


### 収集ルール

- **収集期間**: 直近 **14日間**（初学者は毎日ニュースを追わないため、旬の定義を緩めに設定）
- **URL 重複除去**: 同一 URL の記事は自動で排除
- **ソート順**: Tier 昇順（Tier 1 優先）→ 同一 Tier 内は新しい記事順
- **User-Agent**: `Tech0-GiverBot/1.0`、ソース間にディレイを挟んで礼儀正しく取得

## スコアリング方針

### 対象ペルソナ

スコアリングは以下のペルソナを前提に行う。


| 項目    | 内容                                  |
| ----- | ----------------------------------- |
| 属性    | 20〜35歳、社会人または学生                     |
| 状況    | Tech0 の説明会・イベント参加済み、LINE 登録済み、受講未決定 |
| 知識レベル | ChatGPT 程度。技術用語は苦手                  |
| 心理    | 「学ぶべきか迷っている」「背中を押してほしい」状態           |


### 4 軸スコアリング（各 0〜10 点、合計 40 点満点）


| 軸            | 採点基準                      | 優先度 |
| ------------ | ------------------------- | --- |
| **学習価値**     | 「これを知ると何かできるようになる」感があるか   | 高   |
| **生活・仕事直結度** | 読者の仕事や日常と関係あるか（自分ごと化できるか） | 高   |
| **発見価値**     | ペルソナにとって初めて知る価値があるか       | 中   |
| **初学者親和性**   | 技術用語なしで噛み砕いて説明できる内容か      | 中   |


合計スコア上位 **10 件** を LINE ドラフトとして採用する。

### 除外基準

以下に該当する記事はスコアに関わらず除外する。

- 専門家向けすぎて初学者に届かないもの（論文・実装詳細系）
- 学びに繋がらないゴシップ・炎上系
- 陳腐化した情報（内容が古くなって今さら役立たない）
- **Tier 2（Qiita/Zenn）の記事**は文体参考のみ。コンテンツとして採用しない

### LINE 文面フォーマット

```
【今日のテック/AIニュース】

📌 [トピック名：15文字以内・キャッチー]

[何が起きたか：2〜3行、平易な言葉・技術用語なし]

💡 あなたへの影響：[1行、仕事や日常との具体的な関連]

#Tech0 #AI #テクノロジー
```

トーンは「平易・前向き・押しつけがましくない」を厳守する。
